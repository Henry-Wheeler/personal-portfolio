# Personal Portfolio

Wii Menu–inspired UI: React, Vite, and a Three.js “About Mii” channel with a rigged Mii-style character.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Public repo note

Do **not** commit API keys, tokens, private URLs, or personal secrets. Use environment variables (see `.gitignore`) if you add server-side or paid APIs later.
